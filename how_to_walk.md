歩ければスキーができるから極めて正しく歩ければスキーはうまくなるへ

歩ければスキーができるっていうのは初代デモ選優勝の平澤さんの考え　じゃあそれをもう一歩進めて、スキーの練習をすることでうまく歩く　うまく歩く練習をすることでスキーが上達するようにできる可能性はあるのかを考えてみる。

スキーと足をひかくすると、プライオメトリクス　SSC　によって弾性力でパワーを貯められる量がスキーの方が断然を大きいのと、貯めるための時間も発揮する時間も長い。したがって、プライオメトリクスを使う感覚を養う取っ掛かりとしては、スキーはとても良い。

歩く時にスキーのブーツが前方向に倒れるフレックスを活かす局面は、厚底高反発シューズを使うやり方とほぼ同じだが、少し違うところは、踵近くに加重中心が移ったときにとても強く力を発揮するところ。
足の横アーチを高くして足の反発力をたかくしたとしても限度があるので、つま先で接地しているときは大きな力を加えることができない。一方、スキーは人間の力を超えて板が硬いので度のタイミングでも全力で踏む事ができる。
スキーではパワーを稼ぐための上体の動きを練習しなければならないので、歩行する時にいくらでも力を加えても良い局面で上体を使った力の発揮を練習する必要がある。それが実行できるのは、加重中心がくるぶしの下まで来た時である。

この時間がとても短いのでスキーや投球などで反発する感覚を身に着けて、瞬間的に上体を曲げる力で重心を鉛直上方へ投げ上げる動作を先に十分練習する必要があります。

# ピラティスのロールアップ、ロールダウンを応用する方法

ロールアップは首から尻尾に向かって動かすけど、足を押す力を発揮するためには尾骨から首へと屈曲させていく。

屈曲するのに重心が上へと移動していくことを納得しにくいが、ブロックを一つづつ積むというピラティスの一般的な説明と同時に基礎として既に積まれているブロックを後傾させながら、積まれていないブロックの影響を関節をフリーにすることで排除することでそれは実現できる。
まずは尾骨の一番下の骨の角度を実現したい体勢にセット頭など上部構造がセットする間にいくら前に行ったとしても尾骨をセットすることを優先。セットできたら下から順番に積む。隣同士の関節の距離は０だとみなしてその下の関節をピッチングアップ方向に振れるまで更にその下の関節を上体前面の筋肉を緊張させて動かす。

この感覚を頭の上まで一気に抜けていくように胸やみぞおち、肩首まで連動させて行っていく。腰の下だけが動いて胸は前に倒れるのではなく、あごやおでこも前に行って力が逃げないようにする。
下から積むことを優先するのだが、下を屈曲して上はフリーにするだけで、上のカウンターバランスで下から積む補助をしてはいけない。

尾骨を下から一つづつ後ろに倒すために胴体前面の筋肉を緊張させることは理解できるが、みぞおちから胸椎へ至るときに背中の筋肉を使ってしまいがちである。
尾骨から頭の先まで後ろに倒す時に体の前面の筋肉のみを使って行うようにする。
胴体を反るのと後ろを倒すのに背中の筋肉を使わないでやる。コレができるのが直感と反する。

# 自転車で練習
これを長周期の適切な負荷でテストする方法としていちばん簡単なのが自転車。
負荷も周期もギアで変えられるしなんかいも繰り返して練習できるのがとても良い。
足首から下はバレエでやるように正しい位置でキープそこから崩さない。
腿を立てて押しに行く局面を下死点の30度手前ぐらいに置く。ストロークが短くなるとタイミングを取りづらくなるけど、腿を寝かせてハムストリングスで押しやすくなると立た敷くなくなる。そのトレードオフを気にしてほしい。
意識としては、腿はただまっすぐ押すだけ。尾骨から頭の上まで最大限丸める場所を上げていき、半身ずつその点が無限遠の下から無限遠の上まで抜けていく感じで実行する。引き上げる足の方は下死点から腸腰筋を緊張させて、上死点から下る局面で一番股関節が前傾していく。
自転車でスピードを出す時は瞬間的な出力が必要ないが、スキーはプライオメトリクスを最大に使うために反らした腰を最大出力する前に丸めていく。

# フリーラインでの練習

後ろ足をターン後半で切り替えの位置まで戻していくる時にどうしてもハムストリングスで支えたまま強引に膝の屈曲で戻しがちである。この股関節の伸展と膝の屈曲を使わずにもとに戻したい。
後ろ足を前足から離さずに腰の鉛直下にできるだけ近くかつ内倒しない位置に置いておく。ポンピングする強度を高めてこのエラーを起こすとハムストリングスがすぐに疲れるのでエラーを認識できる。

# その場で足踏み
ももを上げるときに腸腰筋を緊張させる。ただ歩くようにすると地面についている足の爪先に加重してハムストリングスを緊張させて実行してしまうのでハムストリングスも前腿の筋肉も使わずに尾骨から屈曲させて重心を上に移動させつつ腿を上げるように練習する。
重心を上げきることを最初の目的として練習する。それができたら下げる時に反らす

# 浅いジャンプ
これは十分に習熟したあとに、出力を大きくするために行う。前進しながらジャンプすると着地時に腰をそらす練習と組み合わせることができる。

# 歩く練習
スキーと比べて股関節がエキセントリックに伸長する局面が非常に短い。その瞬間に鉛直上かつ接地点から遠い方向に重心を移動させるために上体前面の緊張を使いたい。歩行は位置エネルギーを利用しないので、すぐに行き足が止まってしまう。
股関節を完全にフリーにして直立より股関節が伸展していきながら上体は丸めていく動きを積極的に行い、前への推進力を稼ぐと同時に反対の股関節の屈曲をタイミングよく行うことがとても難しくなる。

# スキーをやることで厳密になる感覚

仕事をせずに股関節以下をフリーにしたときに大腿骨が前に倒れるのか後ろに倒れるのか？足首関節が前後どちらに回転するのか？膝関節の伸展屈曲がどちらかが問題になる。
それを計算して、上体を丸める動きで体を推進できるようになることが一番小さな一歩。

関節の数が少ないと見做せる鉛直と平行に関節が一直線になる状態での鉛直方向への推進から前方向への推進。一直線にならない状態での推進。
そして、足裏の加重中心の変化に伴う股関節を伸展させながら上体下部からを丸める動作を非常に重点的に練習する。

# 足が後ろに流れたときにピッチングダウンモーメントを稼ぎながら重心を後方に加速する操作をする練習方法

腰を丸めた時に大腿四頭筋が伸びて伸張反射をするのでそれを使って目的を達成する。
このときに、加速時よりも深い屈曲をしてから脊柱の一つ上の骨を乗せていく。


# コブがうまくなる歩き方
股関節が伸展する方向に仕事をする.
足を振り下ろして踵で受ける瞬間の股関節の前後方向の位置を垂直上より後ろにしておく. これが1cmでも前に入ると,上体を丸める方向にのみ仕事をして重心を鉛直上または前方に動かせない。
その場で足踏みするなどして、この感覚が身についたと思ったら次のステップに進む

一方の股関節に乗って上体前面の筋肉で重心を鉛直上かつ前方に移動させ、もう片方の足を腸腰筋で引き上げる動作をしてさらに鉛直上に仕事をしていく。
体の中の感覚で可動域を全部使い切って動的ストレッチに入っていて、腸腰筋から先に筋肉痛になりつつ、階段の昇降や不整地の走行、高速走行が可能になるまで練習する。
難しい局面になってできなくなったら前のステップに戻る。
