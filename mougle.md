# 理想の動作
## フットベット上のCoPの固定
フットベットにの真ん中に足裏の加重中心を固定する。これはカービングにおいても成立するかもしれない理想である。
コブに当たったときにスキーヤーとスキーが完全な剛体ならピッチングダウンモーメントが発生する。フットベットのCoPは前方に移動するがそれを防ぐためにブーツカフを前方に曲げながら股関節の伸展トルクを発生させる。
それと同時に上体の重心位置をスキーの前後方向に操作して、スキーのCoPがトップから３０ｃｍぐらいの位置に来るようにする。前すぎるときは上体を丸めて重心を後ろへ、後ろすぎるときは状態を反らす等の動作で重心を前に移動してから股関節を伸展させる。

この原則を守ってターン前半でロールしていれば推進操作をすることになるし、ターン前半でロールせずに谷スキーの鉛直上に重心をおいていれば減速になる。
ターン後半ではロール角の関係が逆になる。

# 練習方法
コブ斜面を横切ってポンピングトラックとして使用したうえで、フットベット上のCoPの固定を実現する。
下り局面で足を抱え込むことを特に意識する。切り替えのタイミングでトップが接地してくことが重要。

# 上体が突っ込んでいるかどうかの判定基準

コブにトップがぶつかってから、全体のブレーキング方向の加速度が最大になる前の局面について、
フォールライン方向の下半身のブレーキング方向への加速度と上半身のブレーキング方向の加速度を比較した場合、下半身のブレーキング方向への加速度が大きくなるようにするべき。ピッチングダウン角速度については上半身の方が小さくなるべき。

全体のブレーキング方向の加速度はコブの登り角度が最大の局面以前に最大になり、それ以後はコブの頂点との距離によってブレーキングが継続するかが決定する。
細かいコブではすぐにブレーキングを弱めてピッチングダウンに移行しないと次の下り局面で張り付けない。

# ピッチングダウンモーメントを稼ぎきっているかどうかの判定基準

全体のブレーキング方向の加速度が最大になったあとから、ビンディングのトーピースがコブの頂点を通り過ぎる前までについて、
ブレーキング時に体が前に突っ込まない範囲でピッチングダウンする。コブの下り坂にトップとスキーセンターが張り付く局面を作れない場合、ピッチングダウンが足りない。

# トップの正しい刺し方

## トップを当てる角度
スキーのショベルの角度を当てる角度としてそれより鈍角にならないように
これでもだめな場合は壁に対してスキー板を刺す角度を鈍角にする。
トップを上げて刺さらないようにしようと思っても、ロール角を変更せずにピッチングアップするためにテールを下げる余地がない場合がある。その場合はロール角を変化させてソール面の垂直下方向に空間を作って壁に対してスキー板が刺さるのを防ぐこれはスキー板のトップが雪面に対して当たってから判断するのは遅いので当たる前に判断する必要がある。
刺さりすぎる時は傾ける。刺さらなすぎる時はロール角を起こして壁に対して直角にする。股関節を内旋外旋してロール角を決定する。

## ポジション
トップを刺すときには重心を失敗時のピッチングダウンモーメントを相殺する位置にセット。例えばトイレットポジション。
これは間違って前転しないように後傾側に位置の安全マージンを大きくとって設定する。その分ピッチングダウンモーメントを高く保ったまま板が上り坂をトップのショベルが３０cm通過させる。
板が上り坂に乗り上げ始めたらできるだけ早いタイミングで道具を曲げきって道具の剛性によって重心が鉛直上方と山側へ加速するように操作する。
その操作とは重心がトイレットポジションから前方へ移動するまで膝関節をフリーにしておき、狙った位置まで膝関節が伸展したら。尾骨からロールアップの逆操作を行って重心を上後方に移動させる。
膝関節の伸展が足りない場合はスキーが後ろに行くだけの摩擦力を確保して重心を上げずに膝関節が伸びるのを待ってから重心を上げる。


ターンの切り替えはコブに当たった時なので、ターンの切替時のニュートラルポジションもコブに当たる時基準で良いはず。
整地ではターンの切替時が傾斜が一番浅い状態になるが、モーグルにおいては傾斜では切替時かどうかを判断しない。

刺さりそうになった時は膝を屈曲させながら股関節を外足を内旋してロール角を増やす。

## 重心の移動量
重心が鉛直下方向に移動する速度が最大になるようにする。

 # 減速しながらコブを乗り越えるときの操作
加重中心が設計上移動できる前端が登り坂を終えるまで股関節を伸ばして重心をできるだけスキーのCoP鉛直上近くに置いてトップが前に出るのを防ぎながら減速。
乗り越えるときに股関節を伸展しすぎて重心が前に行き過ぎた場合となさすぎた場合に分けて対応方法を解説する
ピッチングダウンモーメントが不足する場合コブを乗り越える前にそれを大きくする必要がある。コブを乗り越えてからピッチングダウンモーメントを稼ぐのは正常系ではない。
ピッチングダウン角速度が大きすぎてスキーと上体がピッチングダウンしてしまう場合、お腹だけを前に倒して胸を起こして全体のピッチングダウンモーメントを保存したままスキーと胸頭が前に突っ込まないようにする。

# トップが抜けるときの操作
加重中心が設計上移動できる前端がコブの頂点から前に来て加重中心が設計上移動できる後端がまだ前に来ていない局面での操作

減速したいのであれば、股関節の屈曲トルクを０にすることで踵加重を継続して重心を鉛直上方に加速させテールのグリップを継続

# トップが抜けたあとに移行する方法
抜けたあとは CoPがトーピース直下になるようにスキーの姿勢を制御。


コブの時間的周期が長いとスキーがコブに当たるまでに時間的余裕があるので、股関節を屈曲させないと重心を下げるのが間に合わない局面がコブの頂点を乗り越えてしばらくしたあとに来る。

その場合、太もも前の筋肉が一番伸展するのが、コブを乗り越えたあとの一番斜度がある局面でくることは直感に反している。
ウェーブでコブを乗り越える練習をする場合、まず下り局面で股関節を屈曲させてコンタクトする練習をするが、コブにスキーのトップが刺さるときに股関節を直角に屈曲できる範囲で屈曲開始をできるだけ遅くして、屈曲開始まではできるだけ伸展させておくべき。

そして、股関節を屈曲させてトップを登り坂に刺す。乗り上げたらすぐに股関節の屈曲トルクを０にして、上り局面でブレーキするために股関節が仕事をせずに、重心が前に出る勢いで膝と股関節が伸展されるように操作する。
登り局面で股関節がかなり伸展するのだが、そこからもう一段階伸びながら膝を屈曲する動きは日常生活では滅多にしない。


# 受けの部分で減速する方法
カービングで加速する時と逆のタイミングでスキーに対して同じ動作をする。
CoPから鉛直上とゴールから反対方向とスキーの進行方向と逆に重心を動かす。
特に頭の位置を遠くに持ってくる事が大事。頭がちょっとでも下がったり前にお辞儀すると減速ができない。
谷側の肩を下げても同じ現象が起きる。 肩は重力に水平まで谷側を上げても良い。
外腰を送って頭を上げる方向に力をもらう。その時に外腰の斜腹筋を緊張させるのではなく、股関節を伸展した状態から腸腰筋を収縮させて外スキーを送った反動をもらう。
腸腰筋を緊張させて外スキーを送ると同時に上体を後ろに倒しつつ重心を上げなければ減速できない。ただ単に上体を丸めて頭を下げてはならない。

これを陸上で練習する方法
脊椎のひねりを骨盤と胸の角度が９０度以上になるまで使う。
重心を上げつつ足首を底屈させ膝関節を伸展させ股関節を伸ばす。
このときの最大負荷は 4Gが0.2秒継続するとしてトレーニングするべき。ドロップジャンプでのプライオトレーニングの最短接触時間より十分に長く、エキセントリックなショック吸収がなされて伸張反射が吸収局面で終わっている時間以上である。

トップがコブを抜け出すとトップによる減速はできなくなるので、ブーツセンターを押して減速する。その時に重心を高く前に移動させることでピッチングダウンモーメントを稼ぐ。
膝が伸びきるまえに股関節を屈曲させてピッチングダウンモーメントを増やすのを停止する。

# コブがとても深い場合の対応方法
ものすごい深いコブを一番難しいラインで滑りたい場合について。

下り局面で重力加速度より速く、板を雪面に向かって下げなきゃいけない場合は深いコブに当たる。
その場合にどのようにしてコンタクトするのか？
更にトップを下げる
股関節を伸展させてスキーのソールを雪面に押し当てる。これは重心を移動するために実施しない。

一番の問題はコブの頂点では雪面と重心の位置を近くしなければならないが、
ピッチングダウンモーメントを維持しながら上体をピッチングダウンして頭を次のコブに突っ込まないことである。
乗り越えるために股関節を屈曲する局面で頭を突っ込まずにピッチングダウンモーメントを維持するには上体を反りながら支持基底点と重心を鉛直方向に離しながらピッチングダウンし続ける必要がある。
上体を反ってピッチングダウンする可動域を確保するためにその前の局面で上体を丸めておく必要がある。
股関節が屈曲した時の自由度が一番小さいのでそこを基準にして上体を一番丸めるタイミングを設定する。
股関節を太ももがお腹につくまで屈曲させたとしても上体の上部を優先して鉛直にして突っ込むのを防ぐ。
それでも吸収しきれない場合は速度を落とす。

トップを刺してピッチングアップモーメントを増してからピッチングダウンモーメントを増す局面に入る。
このときの切り替えのタイミングは、歩行によって練習するべき。[練習方法](how_to_walk.md)

# ポーパスターンとズルドンのどちらが前十字靭帯に負荷をかけるか？

## AIに聞いてみた

### 質問
モーグルでスキーのトップ方向にCoPを移動してトップを曲げることで減速する方法は、トップを曲げないで減速する方法と比較して、膝関節への負担はどのような違いがあるのか？

### 答え

モーグルでの減速方法は、使う力の方向と吸収の仕方が違うため、膝関節への負担の「かかり方」が変わります。結論から言うと、トップを曲げて減速する方法の方が、使い方を誤ると膝（特に前側や靭帯）への負担が大きくなりやすい一方で、適切に使えば衝撃分散には有効です。対して、トップを曲げない方法は比較的膝の剪断ストレスは小さいが、別の部位やタイミングで負担が出やすいです。

① トップにCoPを移して曲げて減速する場合
（いわゆる「前圧でトップを使う」）
特徴
* スキー前方に圧をかけてトップをたわませる
* たわみの反発＋雪面抵抗でスピードを落とす
* モーグルの山の裏やコブ頭で使うことが多い
  膝への影響
* 膝の前方剪断力（脛骨前方移動）が増えやすい
   * → 前十字靭帯への負荷が増加しやすい
* 深い前圧になると
   * 膝屈曲＋前荷重で膝蓋大腿関節（膝の前側）に圧縮ストレス
* タイミングが遅れると
   * コブからの突き上げ＋前圧が重なり急激な負荷増加
     👉 要するに 「攻めの減速」＝コントロールできれば効率的だが、ミスると膝に鋭い負担

② トップを曲げずに減速する場合
（ズラし・吸収・ライン取り主体）
特徴
* スキー全体または足裏中央〜後方で圧を分散
* 横ズレや吸収動作でスピードコントロール
* コブの裏で吸収して減速するイメージ
  膝への影響
* 前後方向の剪断ストレスは比較的小さい
* ただし
   * 横ズレが強いと**回旋ストレス（ねじれ）**が増える
   * 吸収が遅れると大腿四頭筋主導で衝撃を受けやすい
     👉 要するに 「守りの減速」＝ピーク負荷は小さいが、疲労や回旋ストレスが蓄積しやすい

比較まとめ
観点	トップを曲げる	曲げない
減速効率	高い	中〜低
膝前方負担	大きくなりやすい	小さめ
回旋ストレス	比較的小さい	増えやすい
タイミング依存	非常に高い	やや低い
リスクの質	急性的（ミス時に大）	慢性的（蓄積型）
実践的なポイント
* トップを使う場合は → コブに当たる前に圧を作る（受けてから押さない）
* 膝を守るには → 股関節主導で圧を作る（膝だけで前に出ない）
* 安全性重視なら → 吸収＋ライン取りベースに、必要な分だけトップを使う

もしレベルや滑り方（ライン重視・エア重視など）が分かれば、もっと具体的に「どの使い分けが膝に優しいか」まで掘り下げて説明できます。
