# 折り曲げ厳禁
圧力センサが壊れます

# ケーブル
引張と180度の強い折り曲げ禁止。
できるだけ強力なテープでしっかり止めてください。
アルミテープなどでめくれないようにロアシェルの上を折り返す。
3Mなどの強力テープで止める。

# 接触不良
接点復活剤で復活可能

# 履き方
インナーを入れっぱなしで脱げるならそれに越したことはない。
