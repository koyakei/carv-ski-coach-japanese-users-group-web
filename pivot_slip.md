# 目的

なんのために横滑りからの方向転換をするのか？
雪面に対してスキーがどんな状態で接触していたとしても、雪面とスキーの間の圧力の中心を動かすことだけで、スキーを雪面に対しての任意のヨーイング角速度をとれるようになるために練習する。
これを最も低速で練習する手段が横滑りからの方向転換で練習する。

雪面に対してソール面が並行で静止している場合、圧力の中心をトップ及びテール方向に移動しても雪面に対してヨーイングすることはない。
雪面に対してソール面が垂直であれば、圧力の中心を前後似移動してずらした分が雪面に対してヨーイングする運動を起こす。
では雪面に対してソール面が並行で動いている場合に圧力の中心を移動してヨーイング運動を起こすにはどうすればいいのか？
圧力の中心を移動することは、自分の体重を骨を使ってソール面に伝えられるので最も楽に力が出る。
最も効率良いが、どのように力を出したらいいのかが難しい操作に習熟することにより、任意のヨーイング角速度を発揮できるようになる。

# 横滑りとは

横滑りとはフォールラインに対してパラレルで直滑降することの逆概念。
フールラインに向かってスキーヤー全体が進行する。
進行方向と直交に板の前後方向を設定する。
[アルペン姿勢](ski_stance)は維持する

# 方向転換
スキーの前後を入れ替える

# 低速での練習の指針
最も困難で正しく実施できていないことがわかる動作をできるだけ低速で行う。
低速で行うほど安全に実施できる。

# 最も敏感で正確な指標の設計
重心を高くして行うと難しくなる。
ヨーイング角度を大きくすると難しくなる。
## フォールラインから横滑りまで

### 角速度の加速
スキーをフォールラインに対して平行に滑降する方向において、方向転換して横滑りをする場合、どのような条件が最も困難であるか？
- 回旋方向の筋力は使用しない。
- 加重中心と重心の移動とロール角の変化で記述する。

横方向に向いたときに谷側になる足を谷足とする。
加重中心が雪面抵抗の中心だとする。
加重中心を谷足のインエッジのフォールラインに近いスキーの端にできるだけ近づける。

#### 直滑降をする段階で回転モーメントがない場合
谷足側のできるだけフォールラインから遠い方向に転ばない範囲で重心を移動する。
転ばない範囲は重心が地面とソール面が並行になったときの支持基底面となりうる範囲の雪面に対して垂直に上方の範囲。

フォールラインと直角方向に雪面抵抗の中心と重心の距離ができる。雪面抵抗中心が雪面に固定されているとすると回転モーメントが増加しヨーイングする。

### 角速度の減速
回転モーメントを減少させる場合、フォールラインと直角方向の雪面抵抗の中心と重心の距離距離を０にする。
横滑りで停止する場合は抵抗中心と重心をスキー真ん中に戻す。

## 横滑りからフォールラインまで
スキーをフォールラインに対して平行にした場合に下となるスキーの端をトップとする。
トップの逆のテールに雪面抵抗中心を設定する。そのために加重中心をテール側インエッジに出来るだけ近づける。
重心と雪面抵抗中心のフォールラインに対して垂直な距離を離す。
回転モーメントが発生し、トップがフォールラインに向く

# なぜこれが最も難しいか？
支持基底面の直線距離が一番短いのはスキーの横方向である。その方向に重心移動をすると、転ぶ方向に一番角運動量が増す。
転びやすい方向に対して最大限重心移動をすることは、転びにくい方向に対して、最大限以下重心移動するよりも難しい。
前後方向の重心移動も最大でないと、一番スキーの幅が広いトップを利用してヨーイング回転モーメントを増すことができない。
したがって、前後方向の重心移動も最大が要求される。
加重についてもおなじ。
ロール角の正確さも違っていた場合には、左右のスキーが同調せずに動くため、高速走行よりも間違いを検出しやすい。
また、インエッジに加重しながら重心をアウトエッジ方向に動かす時に、逆エッジがもっともかかりやすいので精密なロール操作が求められる。

# 成功失敗判定
重心はフォールラインに近い側のインエッジの上に置く

## 回旋操作をしているか？
回旋操作で方向転換を助けていないことを確かめたい。
回旋操作を方向転換を妨げる方向に行い、それを打ち消すように、加重中心と量の移動を行う。
特に、方向転換始動時から妨げる方向に回旋操作を行うことで、最もヨーイングモーメントが発生しにくい場面での加重中心の移動によるヨーイングができているか確かめる。

## 腹筋だけで操作できているか？
腹筋だけで、加重中心の移動が全てできるはずである。股関節の伸展トルクさえも使わないでバランスを取る。

## 横向きからフォールラインまで
横向きで停止した状態からトップに接するフォールラインと並行な障害物を設定する。そこに侵入せずに、テールを現在位置よりフォールラインと直角かつ後方に移動しながらフォールライン方向にスキーを向ける。
フォールラインと垂直方向の重心移動に関しては、最初のトップと逆向きに重心を運ぶ。最初のトップ方向に進んではだめ。

## フォールラインから横向きまで
フォールラインと垂直方向の重心移動に関しては、横向きになった後にテールが来る方向に移動し続ける。トップの方向にすすんではだめ。

## 膝の屈曲トルクを使わない
膝の屈曲トルクが発生しそうになったら、より重心を前に移動し、加重中心が十分に前方に移動したら
腹筋で重心を後ろに移動させて屈曲トルクなしで前に倒れるのを防ぐ。
伸展トルクについても逆を行う練習をする。

## 連続動作
ヨーイング角速度はクロソイド曲線のようになめらかに変化させる。
フォールライン方向への移動速度は一定にする。

# 正しい上体
進行方向は主に斜面下方向になるので斜面下方向に上体を向ける。
進行方向が両手のストックのグリップの中間に来る。
前傾角がブーツと合っているか確かめる。

# より難しくする方法
## 斜面下方向から引っ張ってもらう
引っ張ってもらっても、斜面下方向に等速で動くように練習する。
特に、斜面下にスキーが向いているとき。



# 設計の思想的側面

正しいとは、型として最も優秀かどうかで判定している。 詳しくは[型の条件](https://docs.google.com/document/d/1CUoHCWBLdrV5HQvU6CXcTHY5ZYgaLD6b-iyHYPc5hz4/edit?usp=sharing)を見てください。

- すぐに間違いと判定できて、
- 無限に難しくすることができ、
- 努力の方向性がはっきりしていることである。

ねじれのない一枚バーンで実施することを前提とする。
ねじれのある斜面で実施する場合は、フォールラインにしたがって移動方向を変化させるか、移動方向を維持するか任意に決める。
