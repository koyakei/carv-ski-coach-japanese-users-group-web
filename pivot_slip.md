# 目的

なんのために横滑りからの方向転換をするのか？
雪面に対してスキーがどんな状態で接触していたとしても、雪面とスキーの間の圧力の中心を動かすことだけで、スキーを雪面に対しての任意のヨーイング角速度をとれるようになるために練習する。
これを最も低速で練習する手段が横滑りからの方向転換で練習する。

# 最大限減速する動作

- スキーの進行方向と逆に進む。
- スキーをずらせば減速するので、スキーの前後方向直角に移動させる。
- スキーの元の進行方向に対して直角にヨーイングする。

これらができれば良い。


雪面に対してソール面が並行で静止している場合、圧力の中心をトップ及びテール方向に移動しても雪面に対してヨーイングすることはない。
雪面に対してソール面が垂直であれば、圧力の中心を前後似移動してずらした分が雪面に対してヨーイングする運動を起こす。
では雪面に対してソール面が並行で動いている場合に圧力の中心を移動してヨーイング運動を起こすにはどうすればいいのか？
圧力の中心を移動することは、自分の体重を骨を使ってソール面に伝えられるので最も楽に力が出る。
最も効率良いが、どのように力を出したらいいのかが難しい操作に習熟することにより、任意のヨーイング角速度を発揮できるようになる。

# 横滑りとは

横滑りとはフォールラインに対してパラレルで直滑降することの逆概念。
フールラインに向かってスキーヤー全体が進行する。
進行方向と直交に板の前後方向を設定する。
[アルペン姿勢](ski_stance)は維持する

# 方向転換
スキーの前後を入れ替える

# 低速での練習の指針
最も困難で正しく実施できていないことがわかる動作をできるだけ低速で行う。
低速で行うほど安全に実施できる。

# 最も敏感で正確な指標の設計
重心を高くして行うと難しくなる。
ヨーイング角度を大きくすると難しくなる。

フォールラインへスキーを向けた状態から方向転換する局面

### 角速度の加速
スキーをフォールラインに対して平行に滑降する方向において、方向転換して横滑りをする場合、どのような条件が最も困難であるか？
- 回旋方向の筋力は使用しない。
- 加重中心と重心の移動とロール角の変化で記述する。

横方向に向いたときに谷側になる足を谷足とする。
加重中心が雪面抵抗の中心だとする。
加重中心を谷足のインエッジのフォールラインに近いスキーの端にできるだけ近づける。

#### 直滑降をする段階で回転モーメントがない場合
谷足側のできるだけフォールラインから遠い方向に転ばない範囲で重心を移動する。
転ばない範囲は重心が地面とソール面が並行になったときの支持基底面となりうる範囲の雪面に対して垂直に上方の範囲。

フォールラインと直角方向に雪面抵抗の中心と重心の距離ができる。雪面抵抗中心が雪面に固定されているとすると回転モーメントが増加しヨーイングする。

赤い線の方向に進むつもりで行う。

### 角速度の減速
回転モーメントを減少させる場合、フォールラインと直角方向の雪面抵抗の中心と重心の距離距離を０にする。
横滑りで停止する場合は抵抗中心と重心をスキー真ん中に戻す。

## 横滑りからフォールラインまで
![図面]({{site.baseurl}}/assets/images/pivot_slip_horizontal_to_fall_line.jpeg)
スキーをフォールラインに対して平行にした場合に下となるスキーの端をトップとする。
トップの逆のテールに雪面抵抗中心を設定する。そのために加重中心をテール側インエッジに出来るだけ近づける。
重心と雪面抵抗中心のフォールラインに対して垂直な距離を離す。
回転モーメントが発生し、トップがフォールラインに向く

# なぜこれが最も難しいか？
支持基底面の直線距離が一番短いのはスキーの横方向である。その方向に重心移動をすると、転ぶ方向に一番角運動量が増す。
転びやすい方向に対して最大限重心移動をすることは、転びにくい方向に対して、最大限以下重心移動するよりも難しい。
前後方向の重心移動も最大でないと、一番スキーの幅が広いトップを利用してヨーイング回転モーメントを増やせない。
したがって、前後方向の重心移動も最大が要求される。
加重についてもおなじ。
ロール角の正確さも違っていた場合には、左右のスキーが同調せずに動くため、高速走行よりも間違いを検出しやすい。
また、インエッジに加重しながら重心をアウトエッジ方向に動かす時に、逆エッジがもっともかかりやすいので精密なロール操作が求められる。

# 成功失敗判定
重心がフォールラインに近い側のインエッジの上に置いてあるか？

## 回旋操作をしているか？
回旋操作で方向転換を助けていないことを確かめたい。
回旋操作を方向転換を妨げる方向に行い、それを打ち消すように、加重中心と量の移動を行う。
特に、方向転換始動時から妨げる方向に回旋操作を行うことで、最もヨーイングモーメントが発生しにくい場面での加重中心の移動によるヨーイングができているか確かめる。

## 腹筋だけで操作できているか？
腹筋だけで、加重中心の移動が全てできるはずである。股関節の伸展トルクさえも使わないでバランスを取る。

## 横向きからフォールラインまで
横向きで停止した状態からトップに接するフォールラインと並行な障害物を設定する。そこに侵入せずに、テールを現在位置よりフォールラインと直角かつ後方に移動しながらフォールライン方向にスキーを向ける。
フォールラインと垂直方向の重心移動に関しては、最初のトップと逆向きに重心を運ぶ。最初のトップ方向に進んではだめ。
フォールラインと平行な障害物を避けることができるようになったら、最初にフォールラインと直角になったスキーの後ろ方向に移動しないと避けることができない方向に障害物の角度を変える。最初から後ろに進むのが難しいので、より最初から後ろに移動する量を大きくする。
横方向の移動速度を均一にして、2m下に行くごとにスキー板の1/3ほど後ろに移動できると合格。

## フォールラインから横向きまで
フォールラインと垂直方向の重心移動に関しては、横向きになった後にテールが来る方向に移動し続ける。トップの方向にすすんではだめ。
フォールラインと平行な障害物を避けることができるようになったら、横向きにフォールラインと直角になったスキーの後ろ方向に移動しないと避けることができない方向に障害物の角度を変える。トップが噛んだ直後に後ろに進むのが難しいので、より最初から後ろに移動する量を大きくする。
横方向の移動速度を均一にして、2m下に行くごとにスキー板の1/5ほど後ろに移動できると合格。

どうやったら停止しや状態から後ろ方向にスピードをつけられるのか？

逆エッジで雪面に引っ掛けてフォールラインと直角にスキーセンターを利用して踏み切る。重心を横に移動させそれに足が引っ張られるようにして回旋によらないヨーイングを実現する。逆エッジのグリップでスノボのパタパタぐらいは推進力を得られるので、そこまで勢いをつけるように。
トップをグリップするとピボットになる。



## 障害物の設定方法
フォールライン下にスキーを向けて、コース端から5cm以内に立つ。
コース外側にスキーを向けて横向きに停止できるように練習する。


## 膝の屈曲トルクを使わない
膝の屈曲トルクが発生しそうになったら、より重心を前に移動し、加重中心が十分に前方に移動したら
腹筋で重心を後ろに移動させて屈曲トルクなしで前に倒れるのを防ぐ。
伸展トルクについても逆を行う練習をする。

## 足首の角度を内外同じにする
これは適切な前後差をつけることを別の定義で述べたことである。
前後差が適切であり、異常系によって無理に揃えてない限りは足首の角度は内外同じである。

## 適切な前後差とはなんであるか？
前後差はない方が良いのか？
 　バイスキーの場合前後差が最もある場合を一方のスキーの横方向にもう一方が並ばない場合と考える。
その場合、加重中心を前後方向の端に動かした場合片足加重に必ずなるので、内足によってエラーをリカバリーしたり、グリップ力を高くする操作は不可能になる。バイスキーの特徴はターン時にエッジを２本つかえることによってこれらを行えることなので、それができないなら、一本のスキーを両足で操作したほうが良い。エッジ長を任意に変更したい場合は長いスキーを履いておいて荷重中心の移動で接している長さを変更すれば良い。
以上の理由により、前後差はない方が良いのだが、人体の構造上前後差を全く無くして滑ることは難しい。ではどの程度前後差が許されるのかを考える？
高低差のある斜面に横向きに立った場合がこれを考える場合に典型例として適切である。上方の脚を畳んで足首の角度が左右一緒だとすると、上方が前方に出る。左右の足首の角度を固定するものであるブーツを履いた場合、左右のスキーの加重中心を前後方向に対して同じ割合で置こうとすると前後差をつけないとできない。
したがって、加重中心の前後方向のバランスが左右のスキーで同じであれば、それは正常な加重中心だと言える。
足裏の加重中心の位置が左右で同じであって、ブーツカフにかかっている力がおなじでありスキーから受ける前後方向の力が同じであれば、加重中心が同じ場所にあるとする。この場合、ロアシェルに対して働く横方向の力は考えないものとする。
ブーツカフにかかる力が同じで足裏の前後方向の加重中心が同じであり、スキーから受ける前後方向の力も同じであれば、足首の角度は同じになる。問題は、左右のスキーに前後差があって横から見て同じ場所に加重中心を持っていきたい場合、外スキーは前から40% 内スキーは前から５０％の位置に加重中心を固定する必要がありそうにみえることである。前後差をなくしていく努力をすれば雪面に対して７５度までロールさせても前後差はせいぜい25cm程度で収まる。横から見て同じ場所に加重中心を持っていきたい場合で外脚に80%の加重をおいて足首の角度を同じにしたい場合、外スキーが脚を前に押す力を内スキーより強くすれば良い。
プルークになると内スキーのほうがズレにくくてスキーに対して垂直に押す力が同じなら内スキーのほうが前に進む力が大きいのでますます外の足首を曲げて対応してしまうので足首の角度が同じではなくなってしまう。ズレがなければ、スキーを垂直に押す力と推進力は比例するはずなので、外足の加重中心を若干前にしても足首の角度を同じにして滑走することは可能なはずだ。
ブーツの足首を曲げすぎると、ロール方向のフレックスが弱くなってスキーが暴れてずれやすくなるので、足首の角度を左右一緒にして曲げずに使うべき選択ができたほうが良い。


## 足首の角度を同じにする判断
外の足首が曲がりすぎるならもっとかかと側に加重中心をおいてブーツカフと平行に加える力を大きくする。
そうすると、スキーがブーツを前に押す力が大きくなって外側の足首が背屈する。
大きい力がかかるターン後半局面において、スキーのトップに加重中心を近くしないとスキーが雪面に嵌入していかなくてずれてしまう場合外の足首が内側の足首より曲がる。ターン前半でトップを雪面に嵌入させたあと、それをカフと並行な加重によって維持できているかどうかを考える。
横滑りですでに谷側の足首が山側より曲がっている人が多い。その場合、膝を屈曲させるか、上体を谷側に向けて重心をスキーの後ろ方向にに移動して、加重中心もうしろに移動する。それを自撮り棒で撮ったりして足首の角度が揃っているかを確認してから、さらに両足裏の加重中心の各々の足先と踵後端の間の前後バランスを同一にする。ブーツ回りが同一になったら、膝の向きをスキーの前後方向に合わせる。その後に腰の向き上体ストックワークを正しくしてから、ピボット操作の練習をする。ピボットをする前に横滑りの以上のチェック項目をクリアするべき。

## 凹凸がある雪面

### 難しくなる条件
雪面の凸凹に逆エッジで引っかかりやすい条件だと難しくなる。

逆エッジで引っかかりやすい例

### どのようにクリアすればよいのか？
グリップしたい箇所を雪面に沈めそれ以外を持ち上げる。
そのために沈めたい瞬間に沈めたい場所に対して雪面に対して垂直に力を加えられるようにする。
それでもだめなら、膝の進展トルクを使ってもいいので、ピッチング方向のモーメントを増やす。
ロール方向もズレが継続する限り大きくする。これをやることによって、ロール角を大きくしても雪面に食い込まない操作を覚えられる。これの逆操作をするとロール角が小さくても雪面に食い込ませされる。
スキーに対して雪面に近い横方向に力を加えてエッジを雪面に食い込ませない。雪面から遠い横方向には力をできるだけ加える。

# 足首から下を動かす練習方法
足首を使う方法をどのようにして練習するか？

重心移動を最小限にしてより膝関節を使わない方法を探す方法。

足首の内反外反、背屈、底屈を利用して、内転外転を利用せずに任意の運動を実現したい。

底屈すると踵荷重、背屈はその逆。
内反だと小指側、外反だと親指側に荷重中心が移動する。

ブーツのカフに寄りかかってブーツに力を加えるために重心移動できる。

足首の運動と、ブーツの前後方向にどう寄りかかるかを組み合わせて任意のヨーイングを生み出す。

例えば、スキーを横にして、ブーツの後ろに寄りかかって、ブーツの前かつ谷側に近い横方向に加重中心を移動させるとどうなるのか挙動を確認する。

# 上下動
どのようなタイミングで上下動をするべきなのか？
圧力を増やすためにピッチング方向の加速度がほしい。
横滑りから板をフォールライン方向に向けるときにピッチングダウンして 踵への圧力を増やす。そのときに重心と雪面との距離が離れていくとやりやすい。
トップに最大の圧力をかけたら、その反動でピッチングアップする。ピッチングアップしているときに重心と雪面との距離を近づける。

## 逆エッジをしてしまう
ボコボコに引っかかって逆エッジしてしまう場合、斜面に対して並行に板を動かす力を大きくすると解消できる。
ではどのようにそれを増やすのか？
デッキ面に対して垂直下方向に力を加えて斜面に対して並行な移動が増える場面で立ち上がる。
具体的にはフォールラインに対して板を横向きにしているときに、谷足インエッジのテール側を雪面に食いこんで動かない範囲で谷方向に重心を移動する。なおかつ板の中心を動かしたい方向である、テール側に向かって重心が移動するように立ち上がる。
